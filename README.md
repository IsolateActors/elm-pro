# elm-pro

> 企业 Vue 开发配置

```
自定义配置：src/config/index.js
rem布局：src/config/rem.js
axios拦截封装：src/api/axios.js
统一请求数据：src/api/data.js
模拟数据同步开发：src/api/mock.js
跨域代理：./vue.config.js
express后台服务: ./server/index.js
```

## Project setup

```
yarn install
```
